# Jonathan Aerts — Portfolio

> Cloud Solutions Architect • Microsoft Azure • IaC / Terraform

Personal portfolio with synthwave/cyberpunk theme, built with Astro 6 + Tailwind CSS v4.

**Live:** [jonathan-aerts.dev](https://jonathan-aerts.dev) (GitHub Pages)

**Resume:** [resume.md](resume.md) — Full CV with experience, certifications, and side projects

## Stack

| Technology | Role |
| --- | --- |
| [Astro 6](https://astro.build) | Static site generator |
| [Tailwind CSS v4](https://tailwindcss.com) | Styling (custom synthwave theme via `@theme`) |
| TypeScript | Typed client scripts |
| [@fontsource](https://fontsource.org/) | Self-hosted fonts (Orbitron, Inter, Sora, Press Start 2P, Rajdhani) |
| GitHub Actions | CI/CD — build & deploy to GitHub Pages |
| GitHub Pages | Public deployment ([jonathan-aerts.dev](https://jonathan-aerts.dev)) |

## Prerequisites

- **Node.js >= 22.12.0** (required by Astro 6)
- npm >= 9.6.5

## Quick Start

```bash
npm install
npm run dev        # http://localhost:4321
```

## Build & Deploy

```bash
npm run build      # static output in /dist
npm run preview    # preview the build locally
```

## CI/CD Pipeline

Push to `main` triggers a GitHub Actions job:

1. **Build** with Node 22 (`npm ci` → `npm run build`)
2. **Deploy** to GitHub Pages via `actions/deploy-pages`
3. Live at [jonathan-aerts.dev](https://jonathan-aerts.dev)

## Pages

| Route | Description | Indexed |
| --- | --- | --- |
| `/` | Landing page — arcade "level select" with animated stars | No |
| `/full` | Full portfolio — synthwave theme, sidebar, all sections | Yes |
| `/recruiter` | Recruiter view — clean, professional, print-friendly | Yes |
| `/curious` | Easter egg — redirects to Google | No |

## Features

### Core

- Multi-page architecture (landing, full portfolio, recruiter, easter egg)
- Synthwave/Miami Vice theme (neon pink + cyan, VHS scanlines)
- Animated particle network (canvas)
- Typing effect cycling through titles
- Responsive sidebar with mobile hamburger menu
- Animated skill bars (IntersectionObserver)

### i18n

- Full FR / EN / JP translations (100+ keys)
- Auto-detect browser language
- Manual language switcher with localStorage persistence

### SEO

- Open Graph + Twitter Cards on all pages (LinkedIn-optimized descriptions)
- JSON-LD Person schema (all pages via BaseHead)
- Sitemap + robots.txt
- Canonical URLs
- Custom OG preview image (1200x627, generated via Sharp)
- `article:published_time` + `meta author` for social previews

### Performance

- Self-hosted fonts via @fontsource (no Google Fonts requests)
- WebP images with PNG/JPG fallback (`<picture>`)
- Panel.png: 1.6MB → 59KB WebP (-96%)
- `width`/`height` on all images (anti-CLS)
- `fetchpriority="high"` on hero images

### Accessibility

- `:focus-visible` outlines on all pages
- `aria-pressed`, `aria-label`, `role="group"` on language buttons
- `aria-hidden` on decorative elements
- `prefers-reduced-motion` support (disables animations + canvas)
- Skip-to-content link (recruiter)
- Semantic `<header>`, `<footer>`, `<main>`
- `<noscript>` fallback on landing page
- WCAG AA contrast ratios

### Security

- Email obfuscation (decoded client-side via JS, invisible to scrapers)
- `rel="noopener noreferrer"` on all external links
- Content Security Policy meta tag (recruiter)

### Analytics & Monitoring

- [GoatCounter](https://goatcounter.com) — privacy-friendly, no cookies, GDPR compliant
- [Google Search Console](https://search.google.com/search-console) — indexing, search performance, sitemap submission
- Dashboard: [john6810.goatcounter.com](https://john6810.goatcounter.com)

### Other

- PWA manifest + multi-size favicons (16/32/180/192/512px)
- `meta theme-color` for mobile browser bar
- Print stylesheet (recruiter page)
